#IDEA 小技巧

  • cmd + shift + enter代码自动补全。
  • cmd + p参数提示。
  • psvm等价于public static void main(String[] args) {}。
  • sout等价于System.out.println();。
  • cmd + alt + t可以在代码块外面添加：
    1. if
    2. if / else
    3. while
    4. do / while
    5. for
    6. try / catch
  • alt + enter 在 new MyClass();后会自动填充为MyClass myclass = new MyClass();。
  • cmd + alt + shift + l格式化代码。
